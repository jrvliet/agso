Open the file in Illustrator or Acrobat Pro.
Top Menu: Effect > Document Raster Effects Settings
Resolution: Other: 600ppi

Top Menu: File > Document Color Mode
CMYK

Top Menu: File > Print
General tab
Media: Size should be set to "Custom", enter size of poster in inches
Options: Make sure "Fit to page" is checked
Color Management
Color Handling: select "Let Illustrator determine colors"
Printer Profile: select "US Web Coated (SWOP) v2"
Rendering Intent: select "Relative Colorimetric"
Make sure "Preserve CMYK Numbers" is checked
Advanced
Overprint and Transparency Flattener Options: Preset should be selected to "Custom"
Click the "Custom" button to call up the Custom dialogue window, and make sure "Line Art and Text Resolution" and"Gradient and Mesh Resolution" are at least 600dpi
Click the button on the lower left of the Print dialogue box
Format for: poster printer
Paper size: set up a custom paper size for the poster
Click the button on the lower left of the Print dialogue box
Printer: poster printer
Color Matching
Make sure "Colorsynch" is checked
Profile: US Web Coated (SWOP) v2
Printer Features
Resolution: 600dpi
Color Mode: Normal Color
Click "Print" when done to return to Print dialogue box


All set? Print that sucker!
